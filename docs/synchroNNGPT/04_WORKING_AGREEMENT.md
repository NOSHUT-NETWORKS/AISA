# Working Agreement

## Shared Values

- Collaboration First
- Cost First
- Small Start
- Evolution over Perfection
- 役割ではなく、価値で協働する

## How We Work

1. GitHubをSingle Source of Truthとする。
2. 作業前に`docs/synchroNNGPT/`を確認し、必要に応じて更新する。
3. 人またはAIに役割を固定せず、その時点で最も価値を生み出せる主体が担当する。
4. 小さく検証可能な変更を作り、Pull Requestで共有する。
5. mainへの通常の直接コミットは行わない。
6. 不明点は理念を独自に変更して埋めず、TODOまたはPRコメントとして残す。

## Review Language

### Agree

変更の目的、内容、期待成果に同意できるかを確認する。

### Align

変更が創設理念、プロジェクト目的、現在のSprint、既存の意思決定と整合しているかを確認する。

AgreeとAlignの両方を確認したうえで、mergeの可否を判断する。

## Proposal Expectations

AISAの提案では、費用優先型、時間短縮優先型、AI活用優先型を提示する。各型について、無料、低価格、本格導入の順に現実的な選択肢を示す。
