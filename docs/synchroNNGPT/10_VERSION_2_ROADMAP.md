# AISA Version 2 / Collective Intelligence Roadmap

- Status: Conceptual Roadmap
- Updated: 2026-07-24

## Version Clarification

このRoadmapのVersion 2は、Collective Intelligenceへ向かうArchitecture GenerationとRoadmap Stageを示す。現在実装済みApplicationのVersion番号を示さず、各機能の実装完了を主張しない。

## Roadmap Stages

### v2.0 — AISA LABO Foundation

- Curated Knowledge Libraryの最小表示
- Fictional/Demo Consultation Cases
- Success/Failure Outcome
- Category、Technologies、Observed Date、Freshness Status
- Similar-case browsingの最小Foundation

Implementation plan: [IWP-0002](11_IWP_0002_AISA_LABO_FOUNDATION.md)

### v2.1 — Evidence Ranking

- Pattern MatchとRelevant EvidenceのConceptual Integration
- Evidence Quality、Freshness、Environment Match、Verification Scoreの考慮
- Explainable temporal context

WeightおよびScoring Algorithmは未決定。

### v2.2 — Success / Failure Analysis

- Successful Approachの傾向把握
- Failed Approachからの学習
- Outcomeの比較可能な構造化

### v2.3 — Insight Discovery

- Rare Knowledgeの発見
- Technology/ProblemのCommonness理解
- Unexpected but relevant Caseとの出会い

### v2.4 — AI Curator

- Consultationに関連する有用なCaseの選定
- Curated-bookstore-like discovery
- Relevant and unexpected knowledge presentation

AI Curatorの権限、Safety、Governanceは未設計。

## Future Architecture Item

VersionおよびImplementation Dateは未割当。
